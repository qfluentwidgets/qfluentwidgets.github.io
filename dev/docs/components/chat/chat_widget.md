### [ChatWidget](https://qfluentwidgets.com/zh/price)

![ChatWidget](/img/components/chart/ChatWidget.png)

`ChatWidget` is a component for rendering chat content, supporting streaming content transmission, rendering mathematical formulas, and highlighting code. It delivers exceptional performance to ensure a smooth content display experience. Currently, it only supports PyQt6/PySide6.
